Airversary

  • ・便名が月日(誕生日)となっているフライトを検索できるサイトです．
  • ・例えば12月31日を選択すると，世界中の航空会社のXX-1231便が表示されます(XXは航空会社の2レターコード)．
  • ・なお1月11日→111便，11月1日→1101便となります．
  • ・サイト名Airversaryは，airとanniversaryと掛け合わせた造語です．
