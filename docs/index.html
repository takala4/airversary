Airversary

  • ・便名が月日（誕生日）となっているフライトを検索できるサイトです．
  • ・例えば，12月31日を検索すると，世界中の航空会社のXX-1231便が表示されます（XXは航空会社の2レターコード）．
  • ・なお，1月11日→111便となり，11月1日→1101便となります．
  • ・サイト名のAirversaryは，airとanniversaryと掛け合わせた造語です．
